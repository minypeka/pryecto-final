# chat bot
este proyecto es un desarrollo de un chat bot el cual busca indivir el problema del cambio climatico atrazez del prosesamiento de imagenes e inteligencia artificial clasifica 
y analiza una imagen que ingreses al chat bot y este respondera indicando (contaminadas y no contaminado)segun sea el caso de la imagen.

# comandos 
-/hola:salud al usuario 
-/clasificar:este comando recibe la imagen del usuario la procesae indica al usuario a que clase pertenese
